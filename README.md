# Monpoke Game

## Building game
Run this at your command line
* MacOS & Linux: ```./gradlew build javadoc```
* Windows: ```.\gradlew.bat build```

## Running game
(preface with './' or '.\' for your OS)
* ```monpoke sample_input.txt```
* ```monpoke < sample_input.txt```
